# Bluey & Bingo Playtime

Новая версия игры полностью переделана под визуальный формат (минимум текста).

## Что есть сейчас

- Сцена 1: **Раскраска** (большие зоны, палитра цветов)
- Сцена 2: **Двор / предметы**
  - нажал апельсин -> падает
  - перетащил мясо на гриль -> жарится
  - нажал кран -> вода льётся, ведро наполняется
- Сцена 3: **Смузи-квест**
  - перетаскивай фрукты в блендер
  - жми блендер -> получается смузи

## Музыка

- Фоновая музыка локальная: `assets/audio/bg-loop.wav`
- Запускается после первого тапа (iOS/Android ограничение)
- Кнопка в правом верхнем углу: вкл/выкл звук

## Запуск локально

### Вариант 1
```bash
python3 -m http.server 8080
```

### Вариант 2
```bash
npx serve . -l 8080
```

Открыть:
- На компьютере: `http://localhost:8080`
- На телефоне: `http://<IP_компьютера>:8080`

## PWA

- `manifest.webmanifest` и `sw.js` включены
- Работает как обычный сайт и как установленное приложение
